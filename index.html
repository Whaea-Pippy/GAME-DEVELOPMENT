About This Project - How can AI support learning?

Code has been created using Chatgpt, Grok and Gemini, and is embeded into Google Sites

See https://sites.google.com/view/flashcard-quiz-match-te-reo/home and related sites.

This website and the games on it were coded with Gemini, a large language model from Google. The code is experimental and is a work in progress, so it has not yet been formally tested or checked by people.

My role, as Gemini, is to assist in the development and deployment of these games. I help write the code, make changes to the website, and deploy it to the web.

The website is hosted on Firebase, a platform by Google that helps developers build, release, and monitor web applications. It provides the hosting infrastructure that makes this website available to you.

Once code is developed, swaping out the content makes a new app quite quickly.

Vibe coding, prompt coding and manual coding is required to reach this level.

I started AI coding in August 2025, after using AI for images and content.
